ILocation app for Student Hack 2021

About Project

We have created an ILocation application in Student Hack 2021. Its main function is to call an ambulance or fire department online. Users register in the system and call us from the system when they need help to save time talking to the operator. And they have the ability to track the help sent to them online. When it comes to iLocation, imagine how much space can be saved by saving a 5-minute call to an operator at the scene of a fire.
When ILocation is almost ready for use, it will not benefit investors, it is a project aimed at creating convenience for the population.
